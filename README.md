# retro_computing

Repository for retro computing projects, with a focus on Commodore 64 BASIC,
reconstructed mystery-program material, experiments, notes, and disk images.

## Structure

- `c64_basic_lab/` — general Commodore 64 BASIC projects, experiments, references, and documentation
- `bytes_brothers_archive/` — reconstructed Bytes Brothers-inspired stories, programs, notes, and disk images

This repository is intentionally separate from dissertation-related work.
